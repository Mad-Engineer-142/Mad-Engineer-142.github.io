CVE Crawler Bot
Impulse CVE Crawler Bot - многофункциональный бот, который дает возможность эффективного отслеживания за изменениями Common Vulnerabilities and Exposures (CVE) и предоставляет пользователям последние обновления CVE в зависимости от выбранных параметров в профиле или при вводе конкретного запроса в боте.

Пользователь может запросить список последних CVE, отфильтрованных по типу уязвимости, по ключевому слову, а также по ID. Бот отвечает сообщением, содержащим список найденных уязвимостей, включая номер CVE и другие данные, описывающие уязвимость. Пользователь может также получать уведомления о новых CVE через оповещения, которые зависят от выбранных параметров в профиле.

Кроме того, бот предоставляет пользователю возможность поиска информации о конкретной уязвимости, введя ее номер CVE. Бот отвечает сообщением, содержащим подробное описание уязвимости, уровень угрозы, дату обнаружения, а также ссылки на источники, где можно найти дополнительную информацию и инструкции по устранению уязвимости.

Преимущества нашего сервиса
    1. Уведомление пользователя о CVE
    2. Возможность регистрации профиля
    3. Поиск CVE по параметрам, установленным в профиле
